## Run (Windows)

1. Activate venv:

```powershell
.\venv\Scripts\Activate.ps1
```

2. Start backend (port 8000):

```powershell
python .\run_backend.py
```

3. Start frontend (port 5000) in another terminal:

```powershell
python .\run_frontend.py
```

Open `http://127.0.0.1:5000/`.
